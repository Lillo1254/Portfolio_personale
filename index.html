CSS3

Linguaggio di stile per la presentazione e la formattazione di contenuti HTML

Permette di realizzare layout responsivi, animazioni avanzate, transizioni fluide e design moderni. Supporta variabili, flexbox, grid e media query per adattare i contenuti a qualsiasi dispositivo.

JAVASCRIPT

Linguaggio di programmazione essenziale per l'interattività nel web

Consente di manipolare il DOM, gestire eventi, effettuare chiamate API asincrone e creare esperienze dinamiche lato client. Fondamentale per applicazioni moderne e Single Page Application.

TYPESCRIPT

Superset di JavaScript che introduce tipizzazione statica

Migliora la qualità del codice grazie al controllo dei tipi, prevenendo errori a runtime. Ideale per progetti complessi e scalabili, è spesso utilizzato in combinazione con framework come Laravel per una totale sicurezza.

MySQL

Sistema di gestione di database relazionali open-source

Utilizza il linguaggio SQL per l’interrogazione e la gestione dei dati. È molto diffuso per applicazioni web grazie alla sua velocità, stabilità e compatibilità con numerosi linguaggi di programmazione come PHP.
